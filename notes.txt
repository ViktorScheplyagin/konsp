Купоны действуют совместно с cart price rules (правила применения скидок на определенные товары в корзине), при соблюдении определенных условий (принадлежность к customer group, certain product amount purchase). Чтобы применить купон к покупке, кастомер должен ввести код купона в корзине или предъявить на кассе вашего магазина.
Способы использования купонов:
	E-mail купоны - рассылка по почте, включая новости, каталоги и объявления.
	Печатные купоны - список кодов купонов может быть отправлен на корпоративный принтер
	Купоны, внутри (аккаунта) магазина, для пользователей мобильных устройств, которые могли бы быстро их отсканировать.

Чтобы создать купон, сначала добавьте cart price rule:
	1. Marketing/Promotions/Cart Price Rule
	2. Add New Cart rule
	3. Добавим правило:
		3.1 Заполняем поля:
			3.1.1 Имя купона
			3.1.2 Комментарий (опционально)
			3.1.3 Active -> "Yes"
			3.1.4 Websites - выбираем вебсайт или стор вью
			3.1.5 Выбираем customer group, к которой будет применяться наш купон
			3.1.6 В выпадающем списке "Coupon", выбираем "Specific Coupon" (Наше правило будет использовать коды купонов, введенные покупателями)
			3.1.7 "Coupon Code " - указываем код, который покупатель должен будет ввести, чтобы к нему применилось данное правило или поставьте флажок "Use Auto Generation" для автоматической генерации этого кода
			3.1.8 "Uses per coupon" - вводим количество раз, сколько может быть использован код купона.
			3.1.9 "Uses per customer" - количество раз, сколько может быть использоват код купона одним и тем же зарегестрированным пользователем, принадлежащим одной из выбранный costomer groups (кроме "NO LOGGED IN" group). Для безлимитного использования оставляем поле пустым.
			3.1.10 Поля "From" и "To " - выбираем календарный период, на протяжении которого будет действовать наша скидка
			3.1.11 "Priority" - приоритет данного правила, относительно других правил, которые активны в это же время. Наивысший приоритет 1
			3.1.12 "Public In RSS Feed" -> "Yes", для применения правила к опубликованныйм RSS каналам
			3.1.13  Самое время сохранить правило, нажав "Save and Continue Edit"
		3.2 Опишем условия, для применения данного правила
			3.2.1 Мы видим первое деволтное выражение, имеющее две жирные линки, при клике на которые, получаем доступ к выпадающему списку возможных значений данной части выражения.
			3.2.2 Нажмите на зеленый плюс в кружке, для добавления нового условного выражения (При различных комбинациях таких выражений, мы можем получить самые разнообразные условные конструкции) и выберите переменную, которая будет проверяться на удовлетворение вводимому далее значению. Правило применится только при удовлетворении всех выражений
			3.2.3 Выберите один из операторов сравнения
			3.2.4 Введите сравниваемов значение
		3.3 Опишем действия, которые будут совершаться при соблюдении всех заданных условий
