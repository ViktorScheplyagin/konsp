----Комментарии и синтаксис
Атрибут title у элемента - при наведении появляется "подсказка" со значением атрибута.

----HTML оформление текста
<s></s> - делает текст зачеркнутым
<u></u> - делает текст подчеркнутым
<b> просто делает текст жирным. 
<strong> используется для выделения важной информации. Например, некоторые браузеры для слепых прочитают такой текст более низким тоном.
Тег <em> используется для создания акцента на подчеркнутой фразе.
Тег <i> просто подчеркивает слова. Например термины, названия и т.д.

----HTML списки
Упорядоченный список. По умолчанию нумеруется арабскими цифрами.
	<ol>
		<li>...</li>
		...
	</ol>
	Атрибуты (тех же эффектов можно добиться в css):
		type - тип нумерации. Значения:
			"A" - нумерация по алфавиту (заглавные)
			"a" - маленькие буквы.
			"I" - римскими цифрами.
			"i" - римские цифры обозначаются маленькими буквами.
		start - указать значение, с которого начнется нумерация списка
		reversed - нумерация с конца.
Маркерованный список
	<ul>
		<li>...</li>
	</ul>
	Атрибуты:
		type - тип маркера. Значения:
			"disc" - черный заполненный кружок. Задан по умолчанию.
			"circle" - пустая окружность.
			"square" - заполненный квадратик
Список определений - в нем сначала отображается термин, потом, с новой строки, с абзаца, определение
	<dl>
		<dt>Термин</dt>
		<dd>Определение</dd>
	</dl>

----Ссылки на сайте
Типы ссылок
	Внешняя - ведет на другой домен
	Внутренняя - на другую страницу того же домена
	Якорь - на участок той же страницы
Атрибуты тега <a>
	target - как и где откроется ссылка. Значения
		"_blank" - ссылка откроется в новом окне браузера
		"_self" - в текущем окне
		"_parent" - в родительском фрейме
		"_top" - отменяет все фреймы и открывается в полном окне браузера.
	rel="nofollow" - поисковая система не будет индексировать данную ссылку.
Создание якоря
	<a id="there"></a> - ставим ссылку на том месте, куда надо будет перейти
	<a href="#there">Сюда</a> - эта ссылка будет вести на то место на странице, где расположена вышеуказанная.
<base> - одиночный тег. Указывается в <head>. Имеет те же атрибуты, что и ссылки. Применяет их ко всем ссылкам на данной странице.

----Основы таблиц
<table>
	<caption>Cap!</caption> - заголовок над таблицей.
	<col> - управление колонками таблицы
	<thead> - Указывает семантически на шапку таблицы. Может быть только 1 на всю таблицу. Всегда будет отображаться 			сверху таблицы.
		<tr>...</tr>
	</thead>
	<tfoot> - Указывает семантически на подвал таблицы. Может быть только 1 на всю таблицу. Всегда будет отображаться 			снизу таблицы.
		<tr>...</tr>
	</tfoot>
	<tbody> - Указывает семантически на тело таблицы. Может быть сколько угодно в таблице.
		<tr>
			<th>CapCell</th> - заглавная ячейка. Текст в ней жирный, выравнивается по центру.
			...
		</tr>
	</tbody>
</table>
Атрибуты <td>
	rowspan - в скольких соседних строках объединить ячейку с данной. 
	colspan - в скольких соседних столбцах объединить ячейку с данной.
	В обоих случаях, на самом деле, ширина/высота данной ячейки становится равной сумме соседних объединяемых.
Тег <col> - если прописать несколько подряд, то каждый будет управлять соответствующей колонкой (которая такая же по счету). Атрибуты
	span - значение означает, на сколько соседних колонок будет распростраться управление.
<colgroup> - разрешается писать как одиночный тег, если используется для применения к, непосредственно, колонкам таблицы. Если служит контейнером для <col>, то нужно его закрывать. Отличается от <col> только тем, что при назначении атрибута rules="groups" тегу <table>, браузер будет рисовать границу только между колонками, к которым относится, непосредственно, <colgroup>
